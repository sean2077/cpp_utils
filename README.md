# cpp_utils

This is a C++ utility library that includes some commonly used features, such as file system operations and printing tools.

## Installation

This project uses [CMake](https://cmake.org/) for building. Please make sure CMake is installed on your system.

```sh
# Clone the repository
git clone https://github.com/zhangxianbing/cpp_utils.git

# Enter the project directory
cd cpp_utils

# Create and enter the build directory
mkdir build && cd build

# Build the project
cmake ..

# Compile the project
make
```

## Testing

This project uses Google Test for unit testing. The test code is located in the [`test/`](command:_github.copilot.openRelativePath?%5B%22test%2F%22%5D "test/") directory.

```sh
# Run tests in the build directory
./cpp_utils_test
```
